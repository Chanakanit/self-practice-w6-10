Test 1: Select #title

วิธีที่ใช้: ใช้ document.getElementById("title") แล้วอ่านค่า textContent

Test 2: Count .note
เป็นการฝึกเรื่อง: เลือก element ด้วย class name
วิธีที่ใช้: ใช้ getElementsByClassName("note") แล้วดู .length

Test 3: Select <li> inside #menu
เป็นการฝึกเรื่อง: เลือก child element และวนอ่านค่า
วิธีที่ใช้: ใช้ getElementsByTagName("li") แล้วแปลงเป็น array → .map(li.textContent)

Test 4: Change text of #title
เป็นการฝึกเรื่อง: การเปลี่ยน textContent ของ element
วิธีที่ใช้: เขียนตรง ๆ title.textContent = "DOM Mastery"

Test 5: Add new <li>
เป็นการฝึกเรื่อง: สร้าง element และต่อเข้า DOM
วิธีที่ใช้: document.createElement("li") → appendChild

Test 6: Remove .warning
เป็นการฝึกเรื่อง: การลบ element ออกจาก DOM
วิธีที่ใช้: querySelector(".warning") → .remove()

Test 7: Change background of .note
เป็นการฝึกเรื่อง: การแก้ style ของหลาย element
วิธีที่ใช้: [...note].forEach(...) แล้วแก้ .style.backgroundColor

Test 8: Toggle #secret
เป็นการฝึกเรื่อง: การควบคุมการแสดงผล (display)
วิธีที่ใช้: เขียน ternary style.display = ... ? "block" : "none"

Test 9: Add class highlight
เป็นการฝึกเรื่อง: จัดการ class ของ element
วิธีที่ใช้: querySelectorAll("#appetizer li") แล้วใช้ .setAttribute("class", "highlight")

Test 10: Create button with event
เป็นการฝึกเรื่อง: สร้าง element + event listener
วิธีที่ใช้: createElement("button") → addEventListener("click", ...) → appendChild

Test 11: Replace text of li

Test 13: Clone #appetizer
เป็นการฝึกเรื่อง: การ clone node
วิธีที่ใช้: cloneNode(true) เพื่อ clone ทั้ง subtree แล้ว append

Test 14: Add attribute
เป็นการฝึกเรื่อง: การเพิ่ม attribute ให้ element

Test 16: Prepend new <li>
เป็นการฝึกเรื่อง: เพิ่ม element ไปยัง list
วิธีที่ใช้: createElement("li") → appendChild (แม้โจทย์บอก prepend แต่คุณใช้ append)
